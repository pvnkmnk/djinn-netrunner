# Product Guidelines: Djinn NETRUNNER

## Prose & Communication
- **Tone:** Friendly, approachable, and encouraging. The documentation and console should make complex tasks feel accessible.
- **Style:** Use helpful explanations while maintaining a "hacker" aesthetic that fits the project's cyberpunk theme.

## Visual Branding & Aesthetic
- **Primary Aesthetic:** Cyberpunk Terminal.
- **Design Tokens:** High-contrast color palettes (e.g., neon green or blue on black backgrounds), monospaced fonts, and scanline or terminal-inspired effects.
- **Console Interface:** The console should feel like a retro-future command center while remaining fully functional and intuitive.

## User Experience (UX) Principles
- **Interactive Overlays:** Use modals and toasts for non-disruptive feedback on user actions (e.g., source added, sync started).
- **Real-time Interaction:** Ensure all actions provide immediate feedback and that log data is streamed live without page refreshes.
- **Responsive Simplicity:** While focused on a terminal look, the UI should be robust and work well on various screen sizes for easy remote monitoring.
